Aplicación Web para #YoProgramo

Mi primer proyecto de programación fue para #YoProgramo, la segunda etapa de Argentina Programa 4.0, el objetivo era crear mi propio portfolio web, esta aplicación debia ser de arquitectura distribuida y contener un diseño de interfaz de usuario (front end) que muestre la información, una base de datos que almacene los datos antes mencionados y debia contar con las APIs necesarias para proveer a través de internet la información (back end).
